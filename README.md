# Pressure to Depth Calculator

<a href="https://kaylai.github.io/MagmaDepth/" target="_blank">Use the calculator here</a>

A simple web tool that calculates depth within a planetary substrate from pressure and bulk density using the hydrostatic pressure equation: **h = P / (rho * g)**.

## Features

- Pressure input in bars, kbars, GPa, or kPa
- Density input in g/cm3 or kg/cm3
- Gravitational acceleration presets for all solar system planets and the Moon, plus a custom input option
- Results displayed in both km and m
